#### Aritméticos 
```javascript
let sum = 10 + 5;
let diff = 10 - 5;
let prod = 10 * 5;
let quotient = 10 / 5;
let remainder = 10 % 5;
```
#### Comparación
```javascript
let isEqual = (10 == "10"); // true
let isStrictEqual = (10 === "10"); // false
let isNotEqual = (10 != 5); // true
let isStrictNotEqual = (10 !== 5); //true
```
#### Lógicos 
```javascript
let and = (True && False); // false 
let or = (True || False); // true
let not = !True; // false
```
