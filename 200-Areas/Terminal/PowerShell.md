[[Terminal]]

```bash
cd # Cambia de directorio | Change Directory
ls # Lista todo el contenido del directorio | List System 
ls -directory | Select-Object -First 40 # Lista n contenido del directorio
cp <.origen> <.destino> # Copiar archivo y pegar | Copy
mv <.origen> <.destino> # Mueve archivo o directorio | Move
rm <file> # Elimina archivo | Remuve
rm -r <directory> # Elimina directorio y todo su contenido
mkdir <directory> # Crea un nuevo directorio | Make Directory
cls # Limpia la pantalla de la terminal | Clear
code . # Abrir directorio con el editor de codigo VSCode
notepad <file.txt> # Abre archivos '.txt' con Block de notas
New-Item -ItemType -Name <file.txt> # Crea un archivos '.txt'
hostname # Muestra el nombre del host del sistema
ipconfig # Muestra la configuracion de la red
ping <direccion_ip> # Verificar conectividad con la direccion IP
echo $env:USERNAME # Muestra el nombre del usuario
tree # Muestra el arbol del directorio
Get-Command # Obtiene todos los comandos disponibles en PowerShell
```
